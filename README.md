## 学习安排
### 第1天 7.2
#### 学习内容
- 什么是变量，常量
- 基本类型 int string
- 多行字符串
- double bool
- 字符串插值
- 类型的显示指明

#### 总结
* 1.创建变量使用 var，创建常量使用 let，最好尽可能使用常量。
* 2.字符串以双引号开头和结尾，但是如果你想让它们跨多行，你应该使用三组双引号。
* 3.Integers包含整数，doubles度表示小数，booleans 表示真或假。
* 4.字符串插值允许您从其他变量和常量创建字符串，将它们的值放入字符串中。
* 5.Swift使用类型推断为每个变量或常量指定一个类型，但是如果需要，可以提供显式类型。
